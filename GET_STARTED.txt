                          QUICK START (3 STEPS)
================================================================================

STEP 1: Install Dependencies
-----------------------------
Open PowerShell in this folder and run:

    npm install

If you get an error about scripts being disabled, run this first:

    Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser


STEP 2: Set Up Database & Products
-----------------------------------
Run these commands:

    npx prisma generate
    npx prisma db push
    npx tsx prisma/seed.ts

OR use the shortcut:

    npm run db:setup


STEP 3: Start the Website
--------------------------
    npm run dev

Your website will open at: http://localhost:3000

                        OPTIONAL CONFIGURATION
================================================================================

STRIPE PAYMENTS (for card payments):
------------------------------------
1. Create account: https://dashboard.stripe.com/register
2. Get test keys: https://dashboard.stripe.com/test/apikeys
3. Add to .env file:
   NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_test_...
   STRIPE_SECRET_KEY=sk_test_...

Test card: 4242 4242 4242 4242, any future date, any CVC


EMAIL NOTIFICATIONS (Gmail example):
------------------------------------
1. Enable 2-Step Verification on Gmail
2. Generate App Password: https://myaccount.google.com/apppasswords
3. Add to .env file:
   EMAIL_SERVER_USER=your-email@gmail.com
   EMAIL_SERVER_PASSWORD=your-16-char-app-password

Note: Site works without email - orders are still created successfully!


ADMIN PASSWORD:
---------------
Change in .env file:
   ADMIN_PASSWORD=your-secure-password

================================================================================
                          TROUBLESHOOTING
================================================================================

❌ "Scripts disabled" error:
   Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

❌ Port 3000 in use:
   Next.js will auto-use 3001, 3002, etc.

❌ Database errors:
   Remove-Item prisma\dev.db
   npx prisma db push
   npx tsx prisma/seed.ts

❌ Stripe not working:
   Use Cash on Delivery (works without setup)
   Add Stripe keys to .env for card payments
